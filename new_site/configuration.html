Typography

All HTML headings, <h1> through <h5> are available.

<h1> h1. Heading 1

<h2> h2. Heading 2

<h3> h3. Heading 3

<h4> h4. Heading 4

<h5> h5. Heading 5

Headings options

Create variants of the headings using .white and .feature. The .white class can be applied to change the text color to white and it can be used for all headings. The .feature class can only be used for the <h4> heading.

<h4 class="feature"> h4. Heading 4

<h1 class="white-text"> h1. Heading 1

File structure

Within the download you'll find the following file structure and contents, logically grouping common assets.

Once downloaded, unzip the compressed folder to see the structure of (the compiled) Sherpa. You'll see something like this:

  sherpa/
  ├── assets/
      ├── starter_template.html
      ├── assets/
      │   ├── css/
      │   │    ├──doc.css
      │   ├── img/

This is the most basic form of Sherpa: These files allow for quick drop-in usage in nearly any web project.

What's included

Sherpa comes equipped with HTML, CSS, and JS for all sorts of things, but they can be summarized with a handful of categories visible in the top navigation.

Docs sections

Scaffolding

Global styles for the body to reset type and background, link styles, grid system, and a simple layouts.

Base Elements

Styles for common HTML elements like typography, tables, forms, and buttons.

Components

Basic styles for common interface components like tabs and pills, navbar, alerts, page headers, and more.

JavaScript

Similar to Components, these JavaScript plugins are interactive components for things like tooltips, popovers, modals, and more.

List of components
